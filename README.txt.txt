Using Qt For Windows:

Download the Qt from the official website using the online or the offline installer.
Create a Qt Project and run it in the release mode.
Go to the C:\Qt\(version)\(compiler)\bin  . e.g. C:\Qt\5.9.1\msvc2013_64\bin
open the terminal here and run :
windeployqt (location of your binary file)
Your distributable exe along with the required dll is automatically made.
